# Cloud Computing

## What is Cloud Computing?

Cloud computing is the delivery of computing resources such as servers, storage, databases, networking, and software over the internet on a pay-as-you-go basis.

## Computing Resources

| Resource | Purpose |
|----------|---------|
| Server | Runs applications |
| Storage | Stores files such as images, videos, PDFs, and backups |
| Database | Stores structured data such as customer details and orders |
| Networking | Connects systems and allows communication |
| Software | Provides ready-to-use applications and services |

## Problems with On-Premises Infrastructure

- High upfront cost
- Hardware maintenance
- Limited scalability
- Resource wastage

## Advantages of Cloud Computing

- On-demand resources
- Scalability
- Elasticity
- Cost efficiency
- Pay-as-you-go pricing

## Real-World Example

Suppose you are building an online shopping application.

- **Server** → Runs the application
- **Storage** → Stores product images
- **Database** → Stores customer details and orders
- **Networking** → Connects users to the application
